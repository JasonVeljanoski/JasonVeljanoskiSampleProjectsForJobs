## DEP (Defect Elimination Platform) for Fortescue 2022-2025 (ENCO) [Outdated Version From Current Existing Product]

• Vue.js (Vue 2, Nuxt.js 2, Typescript, Vuex Store, Pina, Vuetify, Primevue, SASS, Chart.js, Mobile PWA)
• Python (SQLAlchemy, Alembic, FastAPI, Pydantic, Swagger)
• Only Developer for initial MVP development, then transitioned to Team Lead for Stage 2 and 3 implementation.
• Brought project from blank canvas to production securing stage two works

Developed the Defect Elimination Platform (DEP) for Fortescue to standardise and centralise the Root Cause Analy-
sis (RCA) process across all business sites. The web application streamlines the creation of Flash Reports, 5-Why
analyses, and RCA documentation, ensuring consistent data entry and improving knowledge sharing. Key features
include the ability to display Word and PowerPoint documents directly in the browser and automatic email notifica-
tions for Flash Reports. The solution has successfully improved collaboration, communication, and decision-making,
driving widespread adoption across the organisation.

## Catalyst for Fortescue 2023-2025 (ENCO) [Outdated Version From Current Existing Product]

• Vue.js (Vue 2, Nuxt.js 2, Typescript, Vuex Store, Pina, Vuetify, Primevue, SASS, Chart.js, Mobile PWA)
• Python (SQLAlchemy, Alembic, FastAPI, Pydantic, Swagger)
• Lead Developer overseeing a team of two
• Team got project from blank canvas to production with stage two work secured

Developed the Catalyst application for Fortescue to centralise and integrate project data from multiple systems,
eliminating data silos and providing a unified platform for tracking stakeholders, risks, work orders, expenses, and
more. Designed to support senior leaders, the application consolidates project details, offering tools like Gantt charts
and risk matrices for effective oversight and decision-making. By synchronising with systems such as DCRM, Eureka,
and BMS ITR, Catalyst enables real-time updates, allowing workflows to continue operating through their native
systems while keeping all data centralised in Catalyst, ultimately streamlining project management.

Developed the ACE web application for Fortescue to centralise and streamline task management for on-site engineers
